Implement Role-Based Access Control (RBAC)

Owner: IT Security Team Effort: 40 hours
  • Define organizational roles and minimum necessary permissions for PHI access.
  • Map existing user accounts to newly defined roles.
  • Integrate an Identity Provider (e.g., Auth0, Okta) to automate role provisioning.
  • Configure 'Least Privilege' policy templates for cloud infrastructure.
  • Audit current access lists and remove all redundant or ghost accounts.
